# 🚀 System Resource Monitor

A real-time system monitoring dashboard built using **Python, Flask, and Chart.js**.

## 🔥 Features
- Live CPU & Memory usage tracking
- Real-time process monitoring
- Search & filter processes
- Kill running processes
- Interactive charts
- Dark mode UI

## 🛠 Tech Stack
- Python
- Flask
- HTML/CSS/JS
- Chart.js
- psutil

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
